# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest (main) | yes |

## Reporting a Vulnerability

This project consists of Claude Code command files (Markdown). There is no server infrastructure or user data processing.

If you discover a security issue — for example a command that could cause unintended file system operations or prompt injection risks — please report it **privately**:

1. Do **not** open a public GitHub issue
2. Open a [GitHub Security Advisory](../../security/advisories/new) in this repository
3. Describe the affected command, the risk, and a reproduction case

You will receive a response within 7 days. If the issue is confirmed, a fix will be released as soon as possible.
